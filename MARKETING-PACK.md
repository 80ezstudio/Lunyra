### 5 FAQ Items

**Q: Is Lunyra actually free?**
A: Yes. The core app is completely free with all soundscapes, ambient themes, and breathing guidance included. No in-app purchases or subscriptions.

**Q: Does the audio keep playing when my phone screen is off?**
A: Yes. Audio continues naturally in the background when you lock your screen — handled by Android's media framework. No extra settings or permissions to enable.

**Q: Does Lunyra collect my personal data?**
A: No. Lunyra does not use trackers, analytics, or any form of data collection. Your session data stays on your device. No account is required to use the app.

**Q: Do I need the internet to use it?**
A: No. All soundscapes and ambient visual themes are bundled within the app. Once installed, Lunyra works fully offline.

**Q: What age group is Lunyra suitable for?**
A: Lunyra is designed for adults and young adults looking for a calm focus or sleep aid. It is not directed at children under 13. It is not a medical device and makes no medical claims.

## 6. DEPLOYMENT + VERIFICATION

### Step-by-Step: GitHub Pages (docs/ folder method)

1. **Create a public GitHub repository** (e.g. `lunyra/website`)
2. **Copy all site files** into a `docs/` folder in the repo root
3. **Push:** `git add . && git commit && git push`
4. **Enable GitHub Pages:** Repo → Settings → Pages → Source: `main` branch, `/ (root)` → Save
5. **Wait 2–3 minutes** for the site to build and deploy
6. **Your site will be at:** `https://YOUR-USERNAME.github.io/REPO-NAME/`

### Step-by-Step: Custom Domain (lunyra.app)

1. Purchase domain from a registrar (Namecheap, Cloudflare, Google Domains)
2. In GitHub repo → Settings → Pages → enter `lunyra.app` as custom domain
3. Add a `CNAME` file to the repo root (content: `lunyra.app`)
4. In your registrar DNS settings:
   - Add **four A records** pointing to GitHub's IPs:
     `185.199.108.153`
     `185.199.109.153`
     `185.199.110.153`
     `185.199.111.153`
   - Add a **CNAME record** for `www` → `YOUR-USERNAME.github.io.`
5. Enable **Enforce HTTPS** in GitHub Pages settings (after DNS propagates, usually 24–48 hours)
6. Update `sitemap.xml` and `index.html` canonical URL to use `https://80ezstudio.github.io/lunyra/`
